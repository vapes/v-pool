# Русский бильярд 🎱

Браузерная игра «Русская пирамида» на TypeScript и PixiJS. Оптимизирована под мобильные устройства (портретная ориентация).

## [Играть](https://vapes.github.io/v-pool/)

## Управление

- **Прицеливание** — нажмите на биток и тяните палец в противоположную сторону от желаемого направления удара
- **Сила удара** — чем дальше оттянете, тем сильнее удар (шкала слева)
- **Винт** — нажмите на индикатор в правом нижнем углу, чтобы задать точку удара по битку (верхний/нижний/боковой винт)
- **Линия прицела** — пунктирная линия показывает предполагаемую траекторию

## Правила

- Русская пирамида: 15 белых шаров + биток
- Шары 68 мм, лузы 72–73 мм (едва больше шара)
- Игра на двоих, ход переходит при промахе или фоле
- Первый до 8 очков побеждает
- Фол: биток в лузе или отсутствие касания

## Разработка

```bash
npm install
npm run dev
```
